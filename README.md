# Meus Sites Favoritos

Projeto desenvolvido em HTML para a atividade prática da unidade sobre desenvolvimento web.

## Descrição

A página apresenta uma lista com alguns dos meus sites favoritos, contendo:

- Nome do site
- Imagem ilustrativa
- Link para acesso

O projeto foi desenvolvido utilizando apenas HTML e CSS básico.

## Sites utilizados

- AniList
- Cifra Club
- Spotify
- Curso em Vídeo
- GitHub

## Tecnologias utilizadas

- HTML5
- CSS3

## Objetivo da atividade

Praticar:

- Estrutura básica de uma página HTML
- Inserção de imagens
- Criação de links
- Organização de conteúdo
- Estilização simples com CSS
